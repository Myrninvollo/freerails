Freerails Java - version 0.0.1



Requires jdk 1.4 (beta), which can be downloaded from:
http://java.sun.com/j2se/1.4/

To compile and run:
(1)	Check out the jfreerails package from the CVS.
(2)	Go to the directory below /jfreerails 
(3)	To compile, type: javac jfreerails/RunFreerails.java
(4)	To run, type: java jfreerails/RunFreerails
	(when asked to enter a map filename, try "test.map")
