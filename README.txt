Freerails Java - version 0.0.3

What's new:
*Track is defined in the file track.xml
*You can build stations, tunnels, bridges, double track, and ferry-boats

Work still needed for the trackbuild system:
*Improving the UI of the trackbuild system.
*Drawing the missing graphics in the trackset



Requires jdk 1.4 (beta), which can be downloaded from:
http://java.sun.com/j2se/1.4/

Getting it:
 	Check-out the jfreerails package from the CVS.
OR
	Download a package from the sourceforge page.

To compile and run:
(1)	Go to the directory below /jfreerails 
(2)	To compile, type: javac jfreerails/RunFreerails.java
(3)	To run, type: java jfreerails/RunFreerails
	(when asked to enter a map filename, try "test.map")
